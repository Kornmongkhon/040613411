สมุนไพรไทย 22 ชนิดที่อาจเสี่ยงสูญพันธุ์เร่งปลูกทดแทน

ถั่วดินโคก


ชื่ออื่นๆ : แผ่นดินเย็น (อุบลราชธานี) นมราชสีห์ นมฤาษี ถั่วดินโคก

ชื่อทางวิทยาศาสตร์ : Sophora exigua Craib.

ลักษณะทางพฤกษศาสตร์ : ไม้พุ่มขนาดเล็ก สูง 15-30 เซนติเมตร ลำต้นสั้นมาก ใบประกอบแบบขนนก เรียงสลับ แนบกับพื้นดินเป็นแนวรัศมี ใบย่อย 9-13 ใบ รูปวงรี ผิวใบมีขนละเอียดสีขาว ดอกช่อกระจะ ออกที่ปลายยอด ดอกย่อยจำนวนมาก กลีบดอกรูปดอกถั่ว สีม่วงเข้ม ก้านช่อดอกยาว ผลเป็นฝักรูปขอบขนาน มีขนละเอียดสีขาว มีเมล็ดเดียว

ถิ่นกำเนิด : สำหรับถิ่นกำเนิดของพิษนาศน์นั้นยังไม่มีข้อมูลยืนยันแน่ชัดว่ามีถิ่นกำเนิดดั้งเดิมในบริเวณใดของโลก แต่ในประเทศไทยนั้น สามารถพบขึ้นทั่วไปในภาคเหนือ และภาคอีสาน บริเวณที่เป็นดินทรายในป่าผลัดใบ และป่าโปร่งทั่วไป ยังมีการนำมาใช้ประโยชน์ด้านสมุนไพรตามภูมิปัญญาชาวบ้านมาตั้งแต่ในอดีตแล้ว

สรรพคุณ :

ยาสมุนไพรพื้นบ้านจังหวัดอุบลราชธานี ใช้ ราก ฝนกับน้ำดื่ม ช่วยลดไข้ในเด็ก ฝนทาแก้พิษงู(ต้องว่าคาถาด้วย) ต้มน้ำดื่ม บำรุงน้ำนม (กินมากไม่ดี) ราก เหง้า ลำต้น ใบ ฝนทาฝี

ยาพื้นบ้านอีสาน ใช้ ราก ฝนน้ำทา แก้ฝี

เทพทาโร


ชื่ออื่นๆ : การบูร จะไคหอม จะไคต้น จวงหอม จวง พลูต้นขาว มือแดกะมางิง

ชื่อทางวิทยาศาสตร์ : Cinnamomum porrectum Kosterm, Cinnamomum parthenoxylon Meissn และ Cinnamomum glanduliferum Nees.

ลักษณะทางพฤกษศาสตร์ : ไม้ต้น สูง 10 – 30 เมตร เรือนยอดเป็นพุ่ม ทึบ กิ่งอ่อนเกลี้ยงและมักจะมีคราบขาว เปลือกสีเทาอมเขียวหรือสีน้ำตาลคล้ำ แตกเป็นร่องยาวตามลำต้น ใบ เดี่ยว เรียงตรงข้าม แผ่นใบรูปรีแกมรูปไข่ ยาว 7 – 20 เซนติเมตร ปลายแหลม โคนสอบ ดอก สีขาว เหลืองอ่อน ออกเป็นช่อประจุกตามปลายกิ่ง ผลกลมเล็ก สีเขียว

ถิ่นกำเนิด : ประเทศไทย (ภาคใต้) มาเลเซีย อินโดนีเซีย

สรรพคุณ :

ใบ รสร้อน ใช้ปรุงเป็นยาหอมแก้ลม จุกเสียดแน่นเฟ้อ แก้อาการปวดท้อง ขับผายลมได้ดี ขับลมในลำไส้และกระเพาะอาหารให้เรอ เป็นยาบำรุงธาตุ ขับเสมหะ

เปลือก รสร้อน มีน้ำมันระเหย 1-25 % และแทนนิน แก้ลมจุกเสียด แน่นเฟ้อ แก้ปวดท้อง ขับลมในลำไส้และกระเพาะอาหาร บำรุงธาตุ

มะตูมนิ่ม


ชื่ออื่นๆ : มะปิน (ภาคเหนือ) กะทันตาเถร ตุ่มตัง ตูม (ปัตตานี) มะปีส่า (แม่ฮ่องสอน) พะโนงค์

ชื่อทางวิทยาศาสตร์ : Aegle marmelos (L.) Correa.

ลักษณะทางพฤกษศาสตร์ : เป็นไม้ยืนต้นสูง 10-15 เมตร ใบเดี่ยว ออกเรียงสลับ รูปรี หรือรูปใบหอกปลายแหลม โคนมน ขอบหยัก ใบมีกลิ่นหอม ยอดอ่อนกินเป็นผักสดกับน้ำพริกได้ ดอกเป็นสีขาวมีกลิ่นหอมแรงและหอมไกล ออกเป็นช่อตามซอกใบและปลายยอด “ผล” รูปกลมรีคล้ายไข่ ผลอ่อนสีเขียว เปลือกแข็ง เมื่อผลสุกเป็นสีเหลือง และเปลือกผลจะอ่อนและนิ่มใช้นิ้วกดจะบุ๋มทันที ไม่เหมือนเปลือกผลมะตูมทั่วไป จึงถูกเรียกว่า “มะตูมนิ่ม” เนื้อในเป็นสีเหลือง มีเมล็ดจำนวนมาก รสชาติหวานหอมปนฝาดเล็กน้อย ขยายพันธุ์ด้วยเมล็ดและตอนกิ่ง

ถิ่นกำเนิด : อินเดีย และในประเทศไทย

สรรพคุณ :

ผล มีกรดไขมันจำเป็น สารฟลาโวนอยด์ ซึ่งช่วยต้านอนุมูลอิสระ ต้านความชรา ต้านแบคทีเรีย รา ไวรัส และปรสิต เป็นยาอายุวัฒนะ เจริญไฟธาตุ แก้ธาตุพิการ เป็นยาบำรุง ปรับธาตุ

มะหาด


ชื่ออื่นๆ : หาด (ทั่วไป) มะหาดใบใหญ่ (ตรัง) หาดขนุน ปวกหาด (เหนือ) ขนุนป่า

ชื่อทางวิทยาศาสตร์ : Artocarpus lakoocha Roxb.

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดใหญ่ สูง 15-25 เมตร ไม่ผลัดใบ ทรงพุ่มกลม ลำต้นเปลาตรง เปลือกต้นสีน้ำตาลอมแดงถึงน้ำตาลเข้ม ต้นแก่ผิวจะหยาบและแตกเป็นสะเก็ดเล็กๆ ยอดอ่อนมีขนสีน้ำตาลออกแดงหนาแน่น มีน้ำยางสีขาว ใบ เดี่ยวเรียงแบบสลับระนาบเดียว รูปไข่หรือรูปขอบขนาน ปลายใบมน ฐานใบมนหรือแหลมกว้าง อาจเบี้ยวไม่สมมาตร ขอบใบเรียบหรือมีซี่จักเล็กน้อย ใบแก่สีเขียวเข้ม เหนียวคล้ายหนัง ด้านบนมีขนหยาบเล็กน้อย ด้านล่างสีเขียวอมเทา รูปหอกซึ่งหลุดร่วงเร็วและมีขนปกคลุมหนาแน่น กิ่งก้านค่อนข้างอ่อน อ้วน ดอกเป็นช่อกลมแน่นสีเหลืองหม่นถึงชมพูอ่อนแบบช่อกระจุก ดอกแยกเพศร่วมต้น ปลายกลีบดอกหยัก

สรรพคุณ :

เปลือกต้น แก้ไข้

แก่น แก้จุกเสียดแน่น แก้ท้องอืดท้องเฟ้อ ขับลม ผายลม แก้ผื่นคัน แก้ตานขโมย แก้กระษัย ขับโลหิต เป็นยาระบาย ถ่ายพยาธิ

ราก แก้ไข้ แก้กระษัยเส้นเอ็น ขับพยาธิ แก้ไข้เพื่อฝีภายใน แก้พิษร้อน

ปวกหาด ละลายน้ำดื่มขณะท้องว่างแล้วดื่มน้ำดีเกลือตามไป ขับพยาธิในท้องหรือละลายน้ำทาแก้ผื่นคัน

เร่ว


ชื่ออื่นๆ : หมากแหน่ง (สระบุรี) หมากเนิง (อีสาน) มะอี้ หมากอี้ มะหมากอี้ (เชียงใหม่) หน่อเนง (ชัยภูมิ)

ชื่อทางวิทยาศาสตร์ : Amomum xanthioides Wall.

ลักษณะทางพฤกษศาสตร์ : เป็นพืชล้มลุก มีเหง้าหรือลำต้นอยู่ในดิน จัดเป็นพืชสกุลเดียวกับ กระวาน ข่า ขิง ใบมีลักษณะยาวเรียว ปลายใบแหลมและห้อยโค้งลง ก้านใบมีขนาดสั้น ออกดอกเป็นช่อจากยอดที่แทงขึ้นมาจากเหง้า ดอกมีสีขาวก้านช่อดอกสั้น ผลมีขนสีแดงปกคลุม เมล็ดมีสีน้ำตาล เร่วมีหลายชนิด เช่น เร่วหอม เร่วช้าง เร่วกอ ซึ่งเร่วเหล่านี้มีลักษณะต้นแตกต่างกันไป

ถิ่นกำเนิด : เชื่อกันว่าเป็นพืชป่าดั้งเดิมของไทย และพบในแถบเดียวกับที่พบกระวานไทย แต่มีบางรายงานระบุว่า เป็นพืชที่มีถิ่นกำเนิดในแถบตะวันออกกลาง

สรรพคุณ :

ใบ ขับลม แก้ปัสสาวะพิการ แก้โรคเกี่ยวกับไขมัน

ดอก แก้พิษอันเกิดเป็นเม็ดผด ผื่นคัน ตามร่างกาย

ผล รสมันเฝื่อนออกเปรี้ยว แก้ไข้เพื่อดีและเสมหะ แก้ริดสีดวงทวาร แก้ธาตุพิการ แก้ท้องอืดท้องเฟ้อ จุกเสียด แก้ปวดท้อง แก้หืดไอ แก้โลหิตขึ้นเบื้องสูง แก้ไข้สันนิบาต ขับผายลม ทำให้เรอ

เมล็ด ขับลม ขับน้ำนม แก้ปวดท้อง แก้คลื่นเหียนอาเจียน ลดไขมันในเลือด ลดความเป็นพิษของสารที่มีผลต่อตับ

ลำต้นเทียมเหนือดิน แก้คลื่นไส้ อาเจียน

ราก เหง้าหรือลำต้นใต้ดิน แก้หืด ไอ แก้คลื่นไส้ อาเจียน

หัวร้อยรู


ชื่ออื่นๆ : กระเช้าผีมด ปุมเป้า ร้อยรู ตาลิมา (มลายู – ภาคใต้); ดาลูปู; กระเช้าผีมด (สุราษฎร์ธานี)

ชื่อทางวิทยาศาสตร์ : Hydnophytum formicarium Jack.

ลักษณะทางพฤกษศาสตร์ : เป็นไม้พุ่มเกาะอาศัยบนต้นไม้ใหญ่โคนลำต้นขยายเป็นขนาดใหญ่ ผิวด้านนอกหยักเป็นลอนคลื่นสีน้ำตาลอมเทาเลื่อมเป็นมัน เนื้ออวบน้ำหนา ภายในเป็นโพรงมีช่องทะลุถึงกันเป็นที่อยู่ของมดและสัตว์ชนิดอื่น

ถิ่นกำเนิด : เป็นพืชที่มีถิ่นกำเนิดในทวีปเอเชียโดย มีแหล่งกระจายพันธุ์ในประเทศอินเดีย และในภูมิภาคเอเชียตะวันออกเฉียงใต้ตั้งแต่เมียนม่าห์ ถึงอินโดนีเซียรวมถึงในหมู่เกาะแปซิฟิก และนิวกินี เป็น โดยมักจะพบในป่าดงดิบทั่วไป ส่วนในประเทศไทยสามารถพบได้ทั่วทุกภาคของประเทศ

สรรพคุณ :

ว่านหัวร้อยรู หัวบำรุงหัวใจ ขับชีพจร

หัว ใช้เป็นยาขับพยาธิ แก้พิษประดงหรืออาการของโรคผิวหนังที่มีผื่นคันคล้ายผด, ช่วยรักษาปอด (ใช้ผสมกับสมุนไพรชนิดอื่น)

กระทุ่มนา


ชื่ออื่นๆ : กระทุ่มดง (กาญจนบุรี) กระทุ่มนา กระทุ่มน้ำ (กลาง) กาตูม (เขมร-ปราจีนบุรี) ตำ (เขมร-สุรินทร์) ตุ้มแซะ ตุ้มน้อย ตุ้มน้ำ (เหนือ) ถ่มพาย (เลย) ท่อมขี้หมู (สงขลา) ท่อมนา (สุราษฎร์ธานี) โทมน้อย (เพชรบูรณ์) กระท่อมขี้หมู

ชื่อทางวิทยาศาสตร์ : Mitragyna diversifolia Wall. Ex. G. Don.

ลักษณะทางพฤกษศาสตร์ : ไม้ต้นขนาดกลาง ผลัดใบ สูง 8-15 ม. เรือนยอดเป็นพุ่มกลม ใบเดี่ยว เรียงตรงข้าม รูปไข่ ปลายมน โคนมนหรือเว้าเล็กน้อย ขอบเรียบ หูใบระหว่างก้านใบ 1 คู่
ปรากฎชัดตามปลายกิ่ง ช่อดอกแบบช่อกระจุกแน่น จำนวนมาก ออกตามปลายกิ่ง แต่ละช่อมีดอกขนาดเล็ก ไม่มีก้านดอก กลิ่นหอม กลีบเลี้ยงเล็กมาก ติดกันคล้ายรูปถ้วย กลีบดอกสีนวล ติดกันเป็นหลอดยาว

ถิ่นกำเนิด : ไทย อินเดีย จีน พม่า ภูมิภาคอินโดจีน มาเลเซีย

สรรพคุณ :

ใบ แก้ท้องร่วง แก้บิดมูกเลือด ปวดมวนท้อง ฤทธิ์เหมือนใบกระท่อมแต่อ่อนกว่า ใช้แทนกันได้

เปลือกต้น รสฝาดแก้ร้อน รักษาโรคผิวหนังทุกชนิด แก้มะเร็งคุดทะราด แก้บิดมูกเลือด

ขันทองพยาบาท


ชื่ออื่นๆ : ดูกใส (อีสาน) ยางปลอก ยายปลวก ฮ่อสะพานควาย (แพร่ น่าน) ขันทองพยาบาท (ภาคกลาง) ดูกหิน (สระบุรี) ข้าวตาก (กาญจนบุรี) มะดูกเลื่อม (เหนือ) ขันทอง (พิษณุโลก) ขุนทอง (ประจวบคีรีขันธ์) กระดูก (ใต้) ป่าช้าหมอง ดูกไทร ขอบนางนั่ง สลอดน้ำ มะดูกดง ข้าวตาก ขนุนดง เจิง

ชื่อทางวิทยาศาสตร์ : Suregada multiflora (A.Juss) Baill.

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดกลาง สูง 7-13 เมตร ทรงพุ่มแน่นทึบ ลำต้นตรง กิ่งก้านอ่อน กิ่งห้อยลง กิ่งมีขนรูปดาว เปลือกต้นสีน้ำตาลแก่ ผิวบางเรียบ เนื้อไม้สีขาว ใบ เป็นใบเดี่ยว เรียงสลับ รูปขอบขนานแกมรูปหอก เนื้อใบหนาทึบ เหนียว หลังใบเรียบลื่นเป็นมัน ท้องใบเรียบสีอ่อนกว่า ฐานใบรูปหัวใจ ปลายใบเป็นติ่งยาว ขอบใบจักฟันเลื่อย ไม่มีขน มีต่อมใสๆ ขนาดเล็ก ผิวใบด้านล่างมีต่อมสีเหลือง และมีขนรูปดาว หูใบแต่ละคู่เชื่อมกัน ดอกสีเขียวอมเหลืองอ่อน ออกเป็นช่อสั้นๆ ตรงซอกใบ

ถิ่นกำเนิด : เป็นพืชที่มีถิ่นกำเนิดอยู่ในเอเชียตะวันออกเฉียงใต้ แถบประเทศอินเดีย พม่า คาบสมุทรอินโดจีน และคาบสมุทรมลายู

สรรพคุณ :

ราก แก้ลม แก้ประดง แก้พิษในกระดูก แก้โรคผิวหนัง รักษาน้ำเหลืองเสีย

เปลือกต้น แก้โรคตับพิการ แก้ปอดพิการ แก้ลมเป็นพิษ แก้โรคผิวหนัง แก้กลากเกลื้อน รักษามะเร็ง รักษามะเร็งคุด เป็นยาถ่าย เป็นยาระบาย เป็นยาบำรุงเหงือก แก้เหงือกอักเสบ ทำให้เหงือกแข็งแรง ทำให้ฟันทน แก้ประดง ถ่ายน้ำเหลือง แก้พิษในกระดูก ฆ่าพยาธิ แก้โรคเรื้อน คุดทะราด แก้กามโรค

เนื้อไม้ แก้กามโรค แก้น้ำเหลืองเสีย แก้ไข้ แก้โรคผิวหนัง แก้มะเร็งคุดทะราด แก้กลากเกลื้อน แก้ลมพิษ แก้ประดงผื่นคัน แก้ประดง แก้พิษในกระดูก ฆ่าพยาธิ แก้โรคเรื้อน แก้ลมและโลหิตเป็นพิษ เนื้อไม้ มีพิษทำให้เมา ใช้เป็นยาเบื่อ

ลำต้น แก้ลมเป็นพิษ แก้พิษต่างๆ แก้ลมพิษ แก้พิษในกระดูก แก้โรคประดง แก้โรคผิวหนัง ฆ่าพยาธิผิวหนัง แก้โรคมะเร็ง คุดทะราด แก้กามโรค ฆ่าพยาธิ แก้โรคเรื้อน แก้กลากเกลื้อน ฆ่าพยาธิโรคเรื้อน

จุกโรหินี


ชื่ออื่นๆ : กล้วยไม้ โกฐพุงปลา พุงปลาช่อน เถาพุงปลา นมตำไร บวบลม

ชื่อทางวิทยาศาสตร์ : Dischidia major.

ลักษณะทางพฤกษศาสตร์ : เป็นไม้เลื้อย อาศัยเกาะตามต้นไม้อื่น ต้นมีน้ำยางสีขาว ใบเดี่ยว ค่อนข้างกลม ก้านสั้นเป็นใบหนา อวบน้ำ ผิวใบทั้งสองด้านมีลักษณะโป่งเป็นถุงเพราะมีมดเข้าไปอาศัย และได้ธาตุอาหารจากของเสียของมด ดอกช่อ ดอกย่อยขนาดเล็กรูปโคม ผลเดี่ยวเป็นฝักเรียว มีขนเป็นพู่ที่ปลาย

ถิ่นกำเนิด : มีถิ่นกำเนิดในประเทศอินเดีย พม่า ไทย อินโดจีน มาเลเซีย ปาปัวนิวกินี ฟิลิปปินส์ไปจนถึงทวีปออสเตรเลีย สามารถพบได้ตามป่าดงดิบทั่วไป ป่าชายเลน ป่าแพะและป่าเบญจพรรณ

สรรพคุณ :

ทั้งต้น ต้มน้ำดื่ม แก้ปวดท้อง เนื่องจากโรคกระเพาะอาหารอักเสบ

ใบ ใช้เป็นยาฝาดสมาน แก้ท้องร่วง แก้ท้องเดิน แก้บิด แก้อาเจียน แก้ปวดเบ่ง มูกเลือด เสมหะผิดปกติ ใช้ภายนอกเป็นยาสมานแผล

ราก แก้ท้องร่วง แก้อาเจียน แก้บิด แก้ปวดเบ่ง แก้เสมหะพิการ สมานแผล แก้ไข้เพื่อโลหิต แก้ลมปลายไข้ ใช้ภายนอกเป็นยาสมานแผล รากนำมาเคี้ยวกับพลู เป็นยาแก้ไอ

ชะเอมไทย


ชื่ออื่นๆ : ชะเอมป่า(กลาง) ตาลอ้อย(ตราด) ส้มป่อยหวาน(พายัพ) อ้อยช้าง(สงขลา) นราธิวาส อ้อยสามสวน (อุบลราชธานี) กอกกั๋น ย่านงาย

ชื่อทางวิทยาศาสตร์ : Albizia myriophylla Benth.

ลักษณะทางพฤกษศาสตร์ : ไม้เถายืนต้นขนาดกลาง สูง 5-8 เมตร มีหนามตามลำต้นและกิ่งก้าน เปลือกนอกสีน้ำตาลอมเทา ใบเล็กละเอียดเป็นฝอย เป็นใบประกอบแบบขนนก 2 ชั้น ยาว 10-15 เซนติเมตร เรียงสลับ ใบย่อยเรียงตรงข้าม ใบย่อยรูปขอบขนาน มีขนที่ขอบใบ ผิวค่อนข้างเกลี้ยง ไม่มีก้านใบย่อย ก้านใบหลัก มีขนหนาแน่น เหนือโคนก้านใบเล็กน้อยมีต่อม ดอกออกเป็นช่อที่ปลายกิ่ง ช่อดอกแบบช่อแยกแขนง ลักษณะเป็นพู่ กลีบดอกสีขาว

ถิ่นกำเนิด : พบมากในทางภาคตะวันออกของไทย ปลูกได้โดยการตอนกิ่งหรือเพาะเมล็ด

สรรพคุณ :

ราก แก้กระหายน้ำ ยาระบาย ขับเสมหะ แก้ไอ เลือดออกตามไรฟัน โรคในลำคอ แก้ลม

ดอก ช่วยย่อยอาหาร ผลขับเสมหะ ใบ ขับเลือดให้ตก เนื้อไม้รสหวาน แก้โรคในลำคอ แก้ลม เลือดออกตามไรฟัน บำรุงกำลัง

เนื้อไม้ แก้โรคในลำคอ แก้ลม เลือดออกตามไรฟัน บำรุงกำลัง

ชิงชี่


ชื่ออื่นๆ : กระดาดป่า (ชลบุรี) ชายชู้ หมากมก (ชัยภูมิ) หนวดแมวแดง (เชียงใหม่) คายซู (อุบลราชธานี) พญาจอมปลวก กระดาดขาว กระโรกใหญ่ กินขี้ จิงโจ้ แสมซอ ค้อนฆ้อง ซิซอ เม็งซอ ราม แส้ม้าทะลาย พุงแก น้ำนอง น้ำนองหวะ เม็งซอ พวงมะละกอ

ชื่อทางวิทยาศาสตร์ : Capparis micracantha.

ลักษณะทางพฤกษศาสตร์ : เป็นพืชในสกุลชิงชี่ ลักษณะเป็นไม้พุ่มหรือรอเลื้อย มีหนาม ใบรูปขอบขนาน รูปใบหอกหรือแกมรูปไข่ โคนใบมนหรือกลม ก้านใบยาว 1-2 เซนติเมตร ดอกออกเป็นช่อสั้น ๆ ตามซอกใบ มี 2-7 ดอก ก้านดอกยาว 0.6-2 เซนติเมตร มีขนสั้นนุ่ม

ถิ่นกำเนิด : มีถิ่นกำเนิดในบังกลาเทศ, จีนตอนใต้และเอเชียตะวันออกเฉียงใต้

สรรพคุณ :

ราก แก้โรคที่เกิดในท้อง ขับลมภายใน แก้ไข้ร้อนทุกชนิด ไข้พิษ แก้โรคตา โรคกระเพาะ รักษามะเร็ง ช่วยให้มดลูกเข้าอู่ และเป็นยาบำรุงหลังคลอดบุตร แก้ไอเนื่องจากหลอดลมอักเสบ แก้หืด

ทั้งต้น ตำพอกแก้ฟกช้ำ บวม แก้ไข้ ขับน้ำเหลืองเสีย

ใบ เข้ายาอาบ แก้โรคผิวหนัง รักษาประดง ใบต้มดื่ม แก้โรคผิวหนัง แก้สันนิบาต ไข้ฝีกาฬ แก้ตะคริว ใบเผาเอาควันสูดแก้หลอดลมอักเสบ ไข้พิษ ฝีกาฬ ไข้สันนิบาต

รากและใบ แก้หืด แก้เจ็บในทรวงอก เป็นยาระงับความร้อน กระทุ้งพิษไข้ออกหัด อีสุกอีใส ตำพอกแก้ฟกช้ำบวม

ดอก แก้มะเร็ง

ผลดิบ แก้โรคในลำคอ เจ็บคอ ลำคออักเสบ

ตับเต่าต้น


ชื่ออื่นๆ : ตับเต่า, เฮื้อนกวาง, มะไฟผี, มะโกป่า (ภาคเหนือ), ตับเต่าใหญ่, เฮื้อนกวง, แฮกวาง, มะมัง, มะเมี้ยง, กากะลา (ภาคอีสาน), เรื้อนกวาง, ชิ้นกวาง, ลิ้นกวาง (ปราจีนบุรี), มะพลับคอ, ตับเต่าหลวง (ราชบุรี)

ชื่อทางวิทยาศาสตร์ : Diospyros ehrelioides Wall.

ลักษณะทางพฤกษศาสตร์ : เป็นไม้ยืนต้นผลัดใบขนาดเล็กถึงขนาดกลาง ทรงพุ่มเป็นรูปกรวยโปร่ง ต้นสูงประมาณประมาณ 10-15 เมตร เปลือกต้นสีน้ำตาลปนเทาหรือสีเทาอมขาว ส่วนเปลือกด้านในเป็นสีน้ำตาลอมแดง ใบออกเป็นใบเดี่ยวมีสีเขียว แบบออกเรียงสลับ มีลักษณะเป็นรูปขอบขนาน รูปวงรี หรือรูปไข่

ถิ่นกำเนิด : มีถิ่นกำเนิดดั้งเดิมในภูมิภาคเอเชียตะวันออกเฉียงใต้ อาทิเช่น พม่า ไทย ลาว กัมพูชา เป็นต้น สำหรับในประเทศไทยสามารถพบได้ในป่าเบญจพรรณ ป่าโปร่ง และป่าเต็งรังทั่วไป ที่มีความสูง 100-450 เมตร จากระดับน้ำทะเลซึ่งจะพบได้เกือบทุกภาคของประเทศยกเว้นทางภาคใต้

สรรพคุณ :

แก่นและราก ใช้เป็นยาแก้วัณโรค ด้วยการนำมาต้มกับน้ำกิน

เปลือก ใช้เป็นยารักษาโรครำมะนาด และยาแก้ท้องร่วง

ราก ใช้ปรุงเป็นยารักษาแผลเรื้อรัง

นางแย้มป่า


ชื่ออื่นๆ : ปิ้งหลาง, ปิ้งขาว, ปิ้งเห็บ (ภาคเหนือ), ขี้ขม (ภาคใต้), ซมซี่, กุ้มคือ (สุโขทัย), ฮอนห้อแดง (เลย), ต่างไก่แดง (ขอนแก่น), พอกวอ, โพะคว่อง (กะเหรี่ยง)

ชื่อทางวิทยาศาสตร์ : Clerodendrum viscosum Vent.

ลักษณะทางพฤกษศาสตร์ : เป็นไม้พุ่มขนาดย่อม มีความสูงของต้นประมาณ 0.5-4 เมตร ลำต้นตั้งเป็นสันสี่เหลี่ยมตรงแต่ไม่มีขนปกคลุม เหมือนนางแย้ง (C.chinense (Osbeck) Mabb.) กิ่งอ่อน และต้นเปราะ ตามลำต้น และกิ่งอ่อนเป็นสีแดง หรือ สีดำอมน้ำตาล ใบเป็นใบเดี่ยว ออกเรียงสลับตามข้อโดยจะออกเป็นคู่ ตั้งฉากกัน ลักษณะของใบเป็นรูปไข่กว้าง หรือ รูปหัวใจ ปลายใบสอบแหลม โคนใบสอบ หรือ เว้ารูปหัวใจส่วนขอบใบจักเป็นซี่ฟันตื้นๆ

ถิ่นกำเนิด : มีถิ่นกำเนิดในประเทศไทยแล้วมีการแพร่กระจายพันธุ์ไปยังบริเวณเขตร้อนใกล้เคียง เช่น อินเดีย ฟิลิปปินส์ อินโดนีเซีย มาเลเซีย เป็นต้น สำหรับในประเทศไทยสามารถพบได้ทั่วทุกภาคของประเทศไทย ตามป่าดิบป่าเบญจพรรณ หรือ ตามชายป่าต่างๆ ที่มีความสูงไม่เกิน 1100 เมตร จากระดับน้ำทะเล รวมถึงที่โล่งที่มีความชื้น หรือ บริเวณใต้ต้นไม้ใหญ่ที่มีพุ่มทึบและมีความชื้นในดินสูง

สรรพคุณ :

ราก ใช้เป็นยาแก้ไข้ ช่วยรักษาลำไส้อักเสบ และเป็นยาช่วยขับถ่ายปัสสาวะ

ใบ เอามาซ้อนกัน 3 ใบ หรือ 7 ใบ แล้วใช้ห่อขี้เถ้าร้อนและใบฮ่อมตำและใบเครือเขาน้ำตำ ใช้ประคบศีรษะแก้อาการปวดศีรษะข้างเดียว

ปลาไหลเผือก


ชื่ออื่นๆ : ตงกัดอาลี (มุสลิม) พญารากเดียว (ทั่วไป) ตุงสอ แฮพันชั้น (ภาคเหนือ) เพียก (ภาคใต้) หยิกบ่อถอง เอี่ยนด่อน (ภาคอีสาน) ไหลเผือก (ตรัง) คะนาง (ตราด) ตรึงบาดาล (ปัตตานี)

ชื่อทางวิทยาศาสตร์ : Eurycoma longifolia jack.

ลักษณะทางพฤกษศาสตร์ : ไม้พุ่มหรือไม้ต้นขนาดเล็ก ลำต้นตั้งตรง สูง 1-10 เมตร เปลือกสีน้ำตาล เป็นไม้ลงราก รากกลมโตสีขาวยาว กิ่งอ่อนมีขนสีน้ำตาล กิ่งก้านสั้นเป็นกระจุกที่ปลายยอดของลำต้น แตกกิ่งก้านน้อย ก้านใบออกจากลำต้นตรงส่วนปลาย เรียงหนาแน่นช่วงปลายกิ่ง ใบประกอบแบบขนนกปลายคี่ เรียงเวียน

ถิ่นกำเนิด : มีถิ่นกำเนิดอยู่ในเอเชียตะวันออกเฉียงใต้ที่มีปริมาณน้ำฝนต่อปีมาก หรือแถบเส้นศูนย์สูตรและเขตร้อนชื้น เช่น ไทย มาเลเซีย อินโดนีเซีย พม่า ลาว กัมพูชา ฯลฯ

สรรพคุณ :

ราก ต้มน้ำดื่ม แก้ไข้ ฝนน้ำดื่ม แก้ปวดท้อง บำรุงโลหิต ราก นำไปเข้ายาบำรุงกำลัง นำรากผสมกับรากโลดทะนงแดง และพญาไฟ ฝนน้ำดื่ม แก้ไข้ ใช้เลิกเหล้า

เจตพังคี


ชื่ออื่นๆ : พังคี ปังคี

ชื่อทางวิทยาศาสตร์ : Croton crassifolius Giesel.

ลักษณะทางพฤกษศาสตร์ : ไม้พุ่มขนาดเล็ก สูงประมาณ 20-40 เซนติเมตร มีรากขนาดใหญ่ใต้ดิน เปลือกรากหลุดออกง่าย มีกลิ่นหอม ใบเดี่ยวเรียงสลับ รูปวงรีหรือรูปไข่แกมรูปหอก โคนใบมน ปลายใบแหลม ขอบใบเรียบ หรือมีฟันเลื่อยเล็กน้อย ผิวใบมีขน ใบหนา แผ่นใบสากคาย ก้านใบยาว 2-3 ซม. ก้านใบและดอกช่อมีขนปกคลุมทั้งต้น ดอกออกเป็นช่อ

ถิ่นกำเนิด : สามารถพบได้ทางตอนใต้ของประเทศจีน และในภูมิภาคเอเชียตะวันออกเฉียงใต้ตามป่าเบญจพรรณที่ระดับความสูงประมาณ 300-400 เมตร

สรรพคุณ :

ราก ต้มน้ำดื่ม หรือฝนทา แก้ท้องอืด ท้องเฟ้อ ในเด็ก หรือตำประคบ แก้ปวด ใช้รักษาโรคไขข้ออักเสบเนื่องจากลมชื้นเกาะติด แก้ปวด เช่น ปวดหลัง ปวดเอว ปวดแข้งขา แก้ฟกช้ำดำเขียว

มะคังแดง


ชื่ออื่นๆ : จงก่าขาว ตะลุมพุกแดง (กาญจนบุรี) ; จิ้งก่าขาว ชันยอด (ราชบุรี); ตุมกาแดง (กลาง) มะคัง (เชียงใหม่) มะคังป่า (กลาง เหนือ) ; มุยแดง ลุมพุกแดง (นครราชสีมา) โรคแดง

ชื่อทางวิทยาศาสตร์ : Gardenia erythroclada Kurz.

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้น กึ่งไม้พุ่ม ขนาดกลาง สูง 6-12 เมตร ใบดกหนาทึบ ลำต้นและกิ่งก้านสีน้ำตาลแดง มีขนนุ่มๆเหมือนกำมะหยี่สีน้ำตาลแดงปกคลุมทั่วไป โคนต้นและกิ่งมีหนามโดยรอบ หนามขนาดใหญ่ พุ่งตรงออกเป็นระยะ ใบเดี่ยว เรียงตรงข้าม รูปวงรีหรือรูปไข่กลับ

ถิ่นกำเนิด : พบได้ตั้งแต่อินเดีย อินโดจีน พม่า และไทย มักขึ้นตามป่าดิบแล้ง ป่าดิบชื้น ป่าเต็งรัง และตามป่าเบญจพรรณ

สรรพคุณ :

เปลือกต้น พบได้ตั้งแต่อินเดีย อินโดจีน พม่า และไทย มักขึ้นตามป่าดิบแล้ง ป่าดิบชื้น ป่าเต็งรัง และตามป่าเบญจพรรณ

ต้น ต้มน้ำดื่ม แก้เลือดลมเดินไม่สะดวก ผสมกับหัวยาข้าวเย็น ต้มน้ำดื่มแก้ไตพิการ (โรคเกี่ยวกับทางเดินปัสสาวะ มีปัสสาวะขุ่นข้น เหลืองหรือแดง มักมีอาการแน่นท้อง กินอาหารไม่ได้) แก้ปวดท้อง ขับพิษโลหิต และน้ำเหลือง

ราก เป็นยาถ่าย ไม่ระบุส่วนที่ใช้ ใช้แก้ไข้

สะค้าน


ชื่ออื่นๆ : ตะค้านเล็ก ตะค้านหยวก

ชื่อทางวิทยาศาสตร์ : 1. Piper interruptum Opiz , 2. Piper ribesioides Wall. 3. Piper wallichii (Miq.) Hand.-Mazz.

ลักษณะทางพฤกษศาสตร์ : เถาแห้งสีน้ำตาล ทรงกระบอก ผิวขรุขระ มีข้อปล้อง หน้าตัดตามขวงมีลายเส้นเป็นแนวรัศมี เปลือกค่อนข้างอ่อน เนื้อสีเหลืองน้ำตาล มีรากฝอยติดอยู่ตามข้อ

ถิ่นกำเนิด : สะค้านพบได้ในทุกภาคของประเทศไทย ที่มีสภาพป่าดิบชื้น โดยพืชสกุลพริกไทยส่วนใหญ่แล้วมีแหล่งที่สำรวจพบอยู่ในเขตร้อนชื้น

สรรพคุณ :

เถา ช่วยขับลมในลำไส้ แก้แน่น แก้จุกเสียด บำรุงธาตุ ทำให้ผายเรอ และใช้ปรุงยาธาตุ แก้ธาตุพิการ เป็นตัวยาประจำธาตุลม

ดอก แก้ลมอัมพฤกษ์

ผล แก้ลมในทรวงอก

ราก แก้ไข้ แก้หืด จุกเสียด รักษาธาตุ

สารภีป่า


ชื่ออื่นๆ : คำโซ่ ตองหนัง ตีนจำ ทำซุง บานมา พระราม โมงนั่ง (อีสาน) ปันม้า แก้มอ้น ฮัก สุน ส้านแดง ส้านแดงใญ่ เหมือดแหล่ สารภี สารภีควาย สารภีหมู (เหนือ) สารภีดอย

ชื่อทางวิทยาศาสตร์ : Anneslea fragrans Wall.

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดเล็กถึงขนาดกลาง ไม่ผลัดใบ สูงถึง 25 เมตร ลำต้นคดงอ แตกกิ่งต่ำ เปลือกต้นเรียบหรือแตกเป็นร่องตื้นไม่เป็นระเบียบ อาจมีรอยแตกลึกเป็นลวดลายละเอียด บางครั้งเปลือกสีครีมค่อนข้างเรียบ เปลือกในสีแดงปนน้ำตาล ใบเดี่ยว

ถิ่นกำเนิด : พบที่จีนตอนใต้ พม่า ภูมิภาคอินโดจีน และคาบสมุทรมลายู ในไทยพบมากทางภาคเหนือ และภาคตะวันออกเฉียงเหนือ พบประปรายทางภาคตะวันตกเฉียงใต้ และภาคใต้ ขึ้นตามป่าเต็งรัง ป่าสนเขา และป่าดิบเขา ความสูง 100–2,000 เมตร

สรรพคุณ :

เปลือก เป็นยาแก้ไข้ ยาแก้บิด ยาถ่ายพยาธิ

ดอก เป็นยาบำรุงหัวใจ ยาแก้ไข้ มีสารช่วยขยายหลอดลม และช่วยขับลม

ผล แก้ลมในทรวงอก

อบเชยไทย


ชื่ออื่นๆ : บอกคอก (ลำปาง), พญาปราบ (นครราชสีมา), กระดังงา (กาญจนบุรี), สะวง (ปราจีนบุรี), ฝักดาบ (พิษณุโลก), กระแจะโมง กะเชียด กะทังนั้น (ยะลา), มหาปราบตัวผู้ อบเชย อบเชยต้น (ภาคกลาง), เขียด เคียด เฉียด ชะนุต้น (ภาคใต้), ดิ๊กซี่สอ กัวเล่ะบิ๊ (กะเหรี่ยง-เชียงใหม่), กะพังหัน โกเล่ เนอม้า (กะเหรี่ยง-กาญจนบุรี), เสี้ยง (ม้ง), ม้าสามเอ็น (คนเมือง), เชียด

ชื่อทางวิทยาศาสตร์ : Cinnamomum spp.

ลักษณะทางพฤกษศาสตร์ : จัดเป็นไม้ยืนต้นขนาดกลางถึงขนาดใหญ่ มีความสูงของต้นประมาณ 15-20 เมตร ทรงพุ่มกลมหรือเป็นรูปเจดีย์ต่ำ ๆ ทึบ เปลือกต้นค่อนข้างเรียบเกลี้ยงเป็นสีน้ำตาลอมเทา เปลือกและใบมีกลิ่นหอม ใบอบเชยไทย ใบเป็นใบเดี่ยวออกตรงข้ามหรือเยื้องกันเล็กน้อย ลักษณะของใบเป็นรูปขอบขนาน

ถิ่นกำเนิด : จะพบได้ในป่าเขาที่ยังอุดมสมบูรณ์หรือป่าดงดิบทั่วไปในประเทศไทย

สรรพคุณ :

เปลือก บำรุงดวงจิต แก้อ่อนเพลีย ชูกำลัง ขับผายลม บำรุงธาตุ แก้บิด แก้ไข้สันนิบาต

รากและใบ ต้มดื่มแก้ไข้จากการอักเสบหลังคลอด เป็นยาขับลม

เฉียงพร้านางแอ


ชื่ออื่นๆ : สีฟันนางแอ (เหนือ), นกข่อ, ส้มป้อง (เชียงใหม่), บงคด (พร), โองนั่ง (อุตรดิตถ์), แก็ก, วงคด, องคต (ลำปาง), แคแห้ง, ต่อไส้, สันพร้านางแอ (กลาง), ร่มคมขวาน (กรุงเทพมหานคร), ขิงพร้า (ตราด, ประจวบคีรีขันธ์), บงมัง (ปราจีนบุรี, อุตรดิตถ์), ม่วงมัง, หมักมัง (ปราจีนบุรี), สีฟัน (ใต้), เขียงพร้านางแอ (ชุมพร)

ชื่อทางวิทยาศาสตร์ : Carallia brachiata (Lour.) Merr.

ลักษณะทางพฤกษศาสตร์ : จัดเป็นไม้ยืนต้นขนาดกลางถึงขนาดใหญ่ สูง 25-30 เมตร ไม่ผลัดใบ ลำต้นตั้งตรง เรือนยอดทรงพุ่มรูปกรวยกว้างทึบ เปลือกสีน้ำตาลอมแดงถึงน้ำตาลอมเทา ผิวเรียบ มีรูอากาศมาก หรืออาจพบเปลือกต้นหนาแตกเป็นร่องลึกตามยาว อาจพบลักษณะคล้ายรากค้ำจุนแบบ prop root เป็นเส้นยาว หรือออกเป็นกระจุกตามลำต้น หรือส่วนโคนต้น ใบเดี่ยว เรียงตรงข้ามสลับตั้งฉาก รูปวงรีแกมรูปขอบขนาน

ถิ่นกำเนิด : ถิ่นกำเนิดดั้งเดิมในเขตร้อนของทวีปแอฟริกาบริเวณเกาะมาดากัสการ์แล้วจึงมีการแพร่กระจายพันธุ์ไปยังเขตร้อนใกล้เคียง

สรรพคุณ :

ลำต้น ฝนน้ำกินแก้ไข้ ต้มน้ำดื่มบำรุงร่างกาย เป็นยาเจริญอาหารสำหรับสตรีหลังคลอด ผสมลำต้นแคด เปลือกต้นตับเต่าต้น ต้มน้ำดื่ม แก้ผิดสำแดง (กินอาหารแสลงไข้ ทำให้โรคกำเริบ อาจมีอาการท้องเสีย)

เปลือกต้น ต้มน้ำอาบรักษาไข้ตัวร้อน

แก่น ช่วยขับลม ระบายความร้อน

เถาเอ็นอ่อน


ชื่ออื่นๆ : ตีนเป็ดเครือ เครือเจน เครือเถาเอ็น เมื่อย หมอนตีนเป็ด หญ้าลิเลน

ชื่อทางวิทยาศาสตร์ : Cryptolepis buchanani Roem. & Schult.

ลักษณะทางพฤกษศาสตร์ : ไม้เลื้อยพาดพันต้นไม้อื่น เปลือกเถาเรียบสีน้ำตาลอมดำ พอแก่เปลือกจะหลุดล่อนออกเป็นแผ่น ทุกส่วนของต้นมี น้ำยางสีขาว ใบ เป็นใบเดี่ยวออกตรงข้ามกัน ใบรูปรี ปลายใบมนมีหางสั้น โคนใบสอบ หลังใบเรียบเป็นมันและลื่น

ถิ่นกำเนิด : เป็นพืชท้องถิ่นของเอเชียตะวันออกเฉียงใต้ รวมถึงประเทศไทย มักพบขึ้นตามป่าราบหรือตามที่รกร้างทั่วทุกภาค แต่มักพบมากทางจังหวัดสระบุรี

สรรพคุณ :

เถา ต้มรับประทานเป็นยาแก้เมื่อย ทำให้เส้นเอ็นหย่อน จิตใจชุ่มชื่น เป็นยาบำรุงเส้นเอ็นในร่างกายให้แข็งแรง แก้เส้นเอ็นพิการ

ใบ ใช้โขลกให้ละเอียด ห่อผ้าทำลูกประคบ ประคบตามเส้นเอ็นที่ปวดเสียวและตึงเมื่อยขบ ทำให้เส้นยืดหย่อนดี

เปราะหอม


ชื่ออื่นๆ : ว่านหอม ว่านตีนดิน ว่านแผ่นดินเย็น หอมเปราะ เปราะหอมขาว

ชื่อทางวิทยาศาสตร์ : Kaempferia galanga L.

ลักษณะทางพฤกษศาสตร์ : จัดเป็นพืชล้มลุก มีอายุราวหนึ่งปี ทั้งเปราะหอมขาวและเปราะหอมแดง เป็นไม้ลงหัวจำพวกมหากาฬ มีลำต้นเป็นหัวอยู่ใต้ดิน หรือที่เรียกว่า “เหง้า” เนื้อภายในของเหง้ามีสีเหลืองอ่อนและมีสีเหลืองเข้มตามขอบนอก และมีกลิ่นหอมเฉพาะตัว มีรสเผ็ดขม

ถิ่นกำเนิด : มีถิ่นกำเนิดในทวีปเอเชียโดยมีเขตการกระจายพันธุ์ตั้งแต่ประเทศอินเดีย ประเทศในเอเชียตะวันออกเฉียงใต้ คาบสมุทรมลายู และชาว รวมไปถึงในจีนตอนใต้และไต้หวัน โดยมักพบในป่าไผ่ ป่าดิบ และป่าผลัดใบ สำหรับในประเทศไทยสามารถพบได้ทุกภาค แต่พบได้มากทางภาคเหนือ

สรรพคุณ :

หัวใต้ดิน รสเผ็ดขม สุมศรีษะเด็ก แก้หวัดคัดจมูก รับประทานขับลมในลำไส้ แก้เสมหะ เจริญไฟธาตุ แก้ลงท้อง แก้โลหิตซึ่งเจือด้วยลมพิษ
